# AppScheduler

AppScheduler is an Android application that allows users to schedule app usage or block apps. It uses an Accessibility Service to monitor and control app access.

## Requirements

- Android device with Accessibility Services support.
- Android Studio for development.

## Setup

1. Clone the repository.
2. Open the project in Android Studio.
3. Build and run the app on your device or emulator.
4. Enable the Accessibility Service in your device settings to allow the app to function correctly.

This app uses Accessibility Services to function. Please ensure you trust the application as it can monitor your screen and interactions.
